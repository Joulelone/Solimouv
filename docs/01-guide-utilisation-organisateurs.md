# Guide d'utilisation - Organisateurs

Ce guide explique comment utiliser la PWA Solimouv' au quotidien.

## 1) Connexion

1. Ouvrir l'application sur `https://votre-domaine`.
2. Cliquer sur `Connexion`.
3. Se connecter avec un compte autorise (Google ou email/mot de passe).

## 2) Installation de la PWA

### Mobile

1. Ouvrir la page d'accueil dans Chrome/Safari.
2. Choisir `Ajouter a l'ecran d'accueil` / `Installer l'application`.
3. Lancer Solimouv' depuis l'icone installee.

### Desktop

1. Ouvrir la page d'accueil dans Chrome/Edge.
2. Cliquer sur l'icone d'installation dans la barre d'adresse.
3. Valider `Installer`.

## 3) Pages publiques (communication)

- `Accueil`: presentation generale du festival.
- `A propos`: mission Up Sport! + contexte Solimouv'.
- `Programme`: ateliers et selection d'activites.
- `Associations`: partenaires et categories d'acteurs.
- `Contact`: canaux de prise de contact.

## 4) Parcours passeport (jour evenement)

1. Le participant ouvre `Mon passeport`.
2. Il montre son QR code au stand.
3. L'animateur ouvre le lien de check-in et valide la participation.
4. La progression se met a jour en temps reel sur le passeport.

## 5) Procedure rapide de verification avant ouverture public

1. Verifier que la page d'accueil charge.
2. Verifier que le menu fonctionne sur mobile et desktop.
3. Verifier que `Programme` et `Mon passeport` s'affichent correctement.
4. Verifier la connexion organisateur.
5. Verifier un cycle complet de check-in.

## 6) Depannage express

- Si l'application ne s'installe pas: verifier HTTPS et manifest.
- Si les donnees ne remontent pas: verifier configuration Firebase.
- Si une page plante: vider le cache navigateur et relancer.
